Nodeとsocket.ioとjQueryとNode_serial_portを利用したwebアプリケーションです。

PSoCとなってますが、ただのシリアル通信です。

↑を押してるときは1の値を返し、
←を押してるときは2の値を返し、
→を押してるときは3の値を返し、
↓を押してるときは4の値を返します。
話したときはすべて0の値を返します。

シリアルポートの設定、ボーレートの設定はsetteing.jsで変更可能です。 あとサイトのポートもsetting.jsで設定可能です。
