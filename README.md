# Node_PSoC_Control<br>

Nodeとsocket.ioとjQueryとNode_serial_portを利用したwebアプリケーションです。<br>

PSoCとなってますが、ただのシリアル通信です。<br>

↑を押してるときは1の値を返し、<br>
←を押してるときは2の値を返し<br>、
→を押してるときは3の値を返し、<br>
↓を押してるときは4の値を返します。<br>
話したときはすべて0の値を返します。<br>

シリアルポートの設定、ボーレートの設定はsetteing.jsで変更可能です。
あとサイトのポートもsetting.jsで設定可能です。
